# Guides

If you are new to the Aptos blockchain, begin with these guides before you get into in-depth development. These guides will help you become familiar with how to develop for the Aptos blockchain using the Aptos SDK.

### [Your First Transaction](guides/first-transaction.mdx)

How to generate, submit and verify a transaction to the Aptos blockchain.

### [Your First NFT](guides/your-first-nft.mdx)

Learn the Aptos `token` interface and how to use it to generate your first NFT. This interface is defined in the [`token.move`](https://github.com/aptos-labs/aptos-core/blob/main/aptos-move/framework/aptos-token/sources/token.move) Move module.

### [Your First Coin](guides/first-coin.mdx)

Learn how to deploy and manage a coin. The `coin` interface is defined in the [`coin.move`](https://github.com/aptos-labs/aptos-core/blob/main/aptos-move/framework/aptos-framework/sources/coin.move) Move module.

### [Your First Fungible Asset](guides/first-fungible-asset.mdx)

Learn how to [deploy and manage a fungible asset](guides/first-fungible-asset.mdx). The `fungible asset` interface is defined in the [`fungible_asset.move`](https://github.com/aptos-labs/aptos-core/blob/main/aptos-move/framework/aptos-framework/sources/fungible_asset.move) Move module.

### [Your First Move Module](guides/first-move-module.mdx)

Write your first Move module for the Aptos blockchain.

### [Your First Dapp](guides/build-e2e-dapp.mdx)

Learn how to build your first dapp. Focuses on building the user interface for the dapp.

### [Your First Multisig](guides/first-multisig.mdx)

Learn how to perform assorted operations using [K-of-N multi-signer authentication](../network/blockchain/accounts.mdx#multied25519-authentication).
