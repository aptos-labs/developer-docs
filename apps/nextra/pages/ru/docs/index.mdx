### Установка

> Примечание: Убедитесь, что у вас установлены `node >= 20` и `pnpm` перед началом.

Чтобы создать новый сайт документации с нуля, перейдите в ваш репозиторий или общий репозиторий документации и выполните:

```sh filename="Terminal"
pnpx @aptos-labs/create-docs
```

Это должно предложить вам ввести название сайта документации и сконфигурирует новый сайт документации Aptos с Nextra.

```sh filename="Terminal"
pnpm install
```

### Разработка

Перейдите к

```sh filename="Terminal"
pnpm dev
```
