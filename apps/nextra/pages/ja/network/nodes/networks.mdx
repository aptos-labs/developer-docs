# Aptosネットワーク

## ネットワーク概要

1. [ローカルネット](http://127.0.0.1:8080) -- コードベースの特定バージョンに対してローカル開発を行うためのスタンドアロンツールで、外部ネットワークは不要です。
2. [デブネット](https://api.devnet.aptoslabs.com/v1/spec#/) -- コミュニティ向けの共有リソースで、データは毎週リセットされ、aptos-coreのメインブランチから毎週更新されます。
3. [テストネット](https://api.testnet.aptoslabs.com/v1/spec#/) -- コミュニティ向けの共有リソースで、データは保持され、ネットワーク構成はメインネットを模倣します。
4. [メインネット](https://api.mainnet.aptoslabs.com/v1/spec#/) -- 実際の資産を扱う本番ネットワークです。

<summary>トークン供給APIの更新</summary>
2024年12月12日、Aptos財団は現在のトークン供給APIを更新しました。このAPIの更新版は、現在の月次期間の開始時点での流通供給量を更新することで、メインネットの現在の流通供給量をより正確に反映しています。以前のバージョンのAPIは、前月の期間の流通供給量を報告していました。ネットワークの流通供給スケジュールに変更はなく、ネットワークの流通供給量は[こちら](https://aptosfoundation.org/currents/aptos-tokenomics-overview)で公開されている元のトークノミクスと一致しています。
