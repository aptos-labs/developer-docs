# パブリックフルノードを実行

独自のパブリックフルノード(PFN)を実行して、Aptosブロックチェーンの状態を同期し、最新の状態を維持することができます。PFNは他のAptosのVFNやPFNと同期することで、ブロックチェーン全体の状態を複製します。PFNは誰でも実行できます。このセクションでは、PFNをデプロイしてAptosネットワークに接続する方法について説明します。ノードの種類の詳細については、[ブロックチェーンの詳細](../blockchain.mdx)セクションで学ぶことができます。

<Callout type="info">
**メインネットへのデフォルト接続**<br />
このドキュメントのデフォルト設定に従うと、PFNはAptosメインネットに接続されます。devnetやtestnetなど、異なるAptosネットワークに接続する場合は、正しいDockerイメージタグを使用するか、バイナリを直接ビルドする場合は適切なソースコードブランチを使用していることを確認してください。

すべてのネットワークのgenesisファイルとwaypointファイルは、こちらで確認できます ➜ https://github.com/aptos-labs/aptos-networks
</Callout>
