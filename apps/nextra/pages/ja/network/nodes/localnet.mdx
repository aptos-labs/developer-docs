# ローカルネットで開発

## 推奨

ほとんどの開発者は、CLIを使用してローカル開発ネットワークを実行することをお勧めします。
単一のローカルネット + フォーセットよりもシンプルで機能が充実しています。
本番環境のネットワークと同じように動作するローカルスタック
([Node API](../../build/apis.mdx) + [Transaction Stream](../../build/indexer/txn-stream.mdx) +
[Indexer API](../../build/indexer.mdx) + フォーセット)が必要な場合は、こちらを選択してください。

- ### [ローカル開発ネットワークを実行](localnet/local-development-network.mdx)

## 特殊な用途

Aptosノードソフトウェア自体を開発している場合や、複雑なテストが必要な場合は、このガイドを参照してください。

- ### [バリデーター付きローカルネットを実行](localnet/run-a-localnet.mdx)
