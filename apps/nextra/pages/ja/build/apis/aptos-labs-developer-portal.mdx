# Aptos Labs API ゲートウェイ

<Callout type="info">
  API ゲートウェイは現在ベータ版です。問題が発生した場合は、[aptos-coreリポジトリでissueを作成](https://github.com/aptos-labs/aptos-core/issues/new?assignees=@geekflyer,@dport,@blakezimmerman&labels=api-gateway&projects=&template=dev_portal_issue.md&title=%5BDev+Portal%5D)して報告してください。
</Callout>

[API ゲートウェイ](https://developers.aptoslabs.com)は、dappを開発するために、Aptos Labsが提供するAPIに簡単かつ迅速にアクセスするためのゲートウェイです。Aptos Labsが運営するポータル（UI）とAPIゲートウェイのセットで構成されています。

API ゲートウェイの詳細については、[API ゲートウェイのドキュメントサイト](https://developers.aptoslabs.com/docs)をご覧ください。
