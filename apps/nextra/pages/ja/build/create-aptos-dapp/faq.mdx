# Create Aptos Dapp よくある質問

## なぜ `import.meta.env` を使用するのですか？

このテンプレートは、開発モードでのみアクセスできるページと、本番モードでもアクセスできるページがあるように構築されています。例えば、「コレクションの作成」と「マイコレクション」ページは開発用であり、開発モードでのみアクセスできます。一方、「パブリックミント」ページは本番モードでもアクセスできます。`import.meta.env` は、dappが開発モードと本番モードのどちらで実行されているかを判断するための `Vite` の方法です。

## dappをライブサーバーに公開しようとしましたが、`404エラー`が発生します

ルートルートの更新が必要かもしれません。サイトを `user-name.github.io/my-repo` にデプロイした場合、ルートルートを `my-repo` に更新する必要があります。

## Tailwind CSSとは何ですか？

Tailwindは、ユーティリティファーストのCSSフレームワークで、コンポーネントのクラス名をスキャンし、ビルド時に対応するスタイルを含む静的なCSSファイルを生成します。

このフレームワークにより、実行時のパフォーマンスに影響を与えることなく、コンポーネントのマークアップと共に配置されたスタイルを素早く作成できます。また、ライトモードとダークモードに対応した一貫したテーマをアプリ全体で維持するのに役立ちます。

Tailwind CSSの詳細については、公式の[ドキュメント](https://tailwindcss.com/docs/utility-first)を参照してください。

## `shadcn/ui`とは何ですか？

Shadcnは、CLIツールを通じてアプリにコピー＆ペーストできるアクセシブルなコンポーネントのコレクションです。ソースファイルはアプリのコードベースに存在するため、必要に応じてカスタマイズできます。

これらのコンポーネントは[Radix UI Primitives](https://www.radix-ui.com/primitives)の上に構築され、[Tailwind CSS](https://tailwindcss.com/)でスタイリングされています。`shadcn/ui`の詳細については、公式の[ドキュメント](https://ui.shadcn.com/docs)を参照してください。

## テーマを変更するには？

このテンプレートのテーマは、`tailwind.config.js`と`frontend/index.css`に分かれています。Tailwindの設定では、すべてのテーマカラー、テキストスタイル、アニメーションキーフレーム、ボーダー半径などを宣言します。ルートCSSファイル（`index.css`）では、ライトモードとダークモードの実際の色の値をCSSカスタムプロパティ（CSS変数）として宣言し、基本の半径値を定義し、必要なグローバルCSSを適用します。

例えば、アプリ内のすべてのボタンとカードをより丸くしたい場合は、`index.css`の基本半径値（`--radius`）を増やすことができます。

新しいテキストスタイルを追加したい場合は、`tailwind.config.js`の最後にある`addTextStyles`関数で定義できます。

また、アプリのプライマリカラーを変更したい場合は、`index.css`で定義されているHSLカラー値を更新できます。

## コンポーネントを追加するには？

追加のコンポーネントは`shadcn-ui` CLIを通じて追加できます。例えば、`Switch`コンポーネントを追加したい場合は、次のコマンドを実行します：

```bash filename="Terminal"
npx shadcn-ui@latest add switch
```

このコマンドにより、スタイル付きのスイッチコンポーネントを含む`switch.tsx`ファイルが`frontend/components/ui`ディレクトリに作成されます。利用可能なshadcnコンポーネントの完全なリストについては、[shadcnコンポーネントのドキュメント](https://ui.shadcn.com/docs/components)を参照してください。

`shadcn/ui`コレクションに含まれていないコンポーネントを追加する必要がある場合は、`frontend/components`の下に独自のコンポーネントを追加するか、作業しているページに特有のものである場合は`frontend/pages`ディレクトリ内に追加することができます。

## 色を追加するには？

独自のカスタムコンポーネントを作成したり、UIに何かを追加したりする場合、新しい色を追加する必要があるかもしれません。新しい色を追加するには、まず`frontend/index.css`でライトモードとダークモードのHSLカラー値を定義し、次に`tailwind.config.js`で定義されているテーマに新しいテーマカラートークンを追加する必要があります。

詳細な手順については、[shadcnのテーマ設定に関するドキュメント](https://ui.shadcn.com/docs/theming)を参照してください。

## ダークモードを追加するには？

dappテンプレートのシンプルさを維持するため、ライトモードのみが設定されています。ただし、テーマには両方のモードの色の値が定義されています。アプリにダークモードを追加したい場合は、shadcnの`ThemeProvider`と`ModeToggle`をアプリに追加するだけです。追加すると、UIはライトモードとダークモードの両方に完全に対応します。これを実現する詳細な手順については、[shadcnのダークモードドキュメント](https://ui.shadcn.com/docs/dark-mode/vite)を参照してください。
