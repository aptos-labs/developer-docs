# create-aptos-dapp

`create-aptos-dapp`は、dapp開発者がAptosネットワーク上でフロントエンドとスマートコントラクトを簡単に作成できるようにするためのテンプレートプロジェクトを構築します。

## なぜcreate-aptos-dappを使用するのか？

- **テンプレート化されたセットアップ**: `create-aptos-dapp`は、事前定義されたエンドツーエンドのdappテンプレートと設定ファイルを生成します。時間がかかり、エラーが発生しやすいプロジェクト構造の手動セットアップを省くことができます。
- **コントラクトディレクトリ:** `create-aptos-dapp`は、Moveスマートコントラクトモジュールの基本構造を含む`contract`ディレクトリを生成します。
- **ベストプラクティス**: `create-aptos-dapp`は、Aptosネットワークの開発におけるベストプラクティスと構造の推奨事項を組み込んでいます。
- **組み込みのMoveコマンド**: `create-aptos-dapp`には、Moveコンパイラの初期化、コンパイル、チェーン上でのスマートコントラクトの公開など、一般的なタスクのためのコマンドが組み込まれています。

## 前提条件

- [node と npm](https://nodejs.org/en) (npm ≥ 5.2.0)
- [Python 3.6+](https://www.python.org/)

## テンプレート

`create-aptos-dapp`は、Aptos上でdappを作成するための設定と美しいUIを備えた、すぐに使えるdappのエンドツーエンドのテンプレートを提供します。

テンプレートの目的は以下の通りです：

1. エンドツーエンドのdappテンプレート例を通じて、ユーザーに様々なAptosスタンダードに慣れてもらうこと。
2. フロントエンド層からスマートコントラクト層まで、Aptos上でのdappの構築方法とその間のすべてについてユーザーを教育すること。
3. シンプルなdappを素早くデプロイするための事前作成されたテンプレートをユーザーに提供すること。

### 現在のテンプレート
現在のすべてのテンプレートは[Aptos Learn](https://learn.aptoslabs.com/en/dapp-templates)で利用可能です。以下の特定のテンプレートについて詳しく読むことができます：

- [ボイラープレートテンプレート](https://learn.aptoslabs.com/en/dapp-templates/boilerplate-template)
- [Telegramミニアプリテンプレート](https://learn.aptoslabs.com/en/dapp-templates/telegram-mini-app-template)
- [NFTミンティングdappテンプレート](https://learn.aptoslabs.com/en/dapp-templates/nft-minting-template)
- [トークンミンティングdappテンプレート](https://learn.aptoslabs.com/en/dapp-templates/token-minting-template)
- [トークンステーキングdappテンプレート](https://learn.aptoslabs.com/en/dapp-templates/token-staking-template)
- [カスタムインデクサーテンプレート](https://learn.aptoslabs.com/en/dapp-templates/custom-indexer-template)
