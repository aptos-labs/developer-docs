# CLI

Aptosコマンドラインインターフェース（CLI）は、Moveコントラクトのコンパイルとテストを支援するツールです。また、Aptosの機能をチェーン上で素早く試すのにも役立ちます。

上級ユーザーの場合、CLIはプライベートAptosネットワークの実行（コードをローカルでテストするのに役立ちます）にも使用でき、ネットワークノードの管理にも役立ちます。
