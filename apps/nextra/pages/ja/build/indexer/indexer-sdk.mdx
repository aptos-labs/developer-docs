# インデクサーSDK

import { IndexerBetaNotice, ThemedImage, Cards, Card, GraphQLEditor } from '@components/index';

<IndexerBetaNotice />

インデクサーAPIは基本的なオンチェーンデータを照会するための強力なツールですが、必要とする正確なデータを常に提供できるとは限りません。
ほとんどの場合、独自のコントラクトをインデックス化したい場合は、インデクサーSDKを使用して独自のカスタムプロセッサーを作成できます。
