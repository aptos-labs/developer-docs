# プールのアクティブな委任者数を取得

`getNumberOfDelegators`は、指定されたプールアドレスのアクティブな委任者数を取得します。このクエリは、ステーキングや委任プールへの参加状況を分析する必要があるプラットフォームにとって特に有用です。

<Callout type="info">
自分で試してみましょう！エディターの下部で変数をカスタマイズできます。
</Callout>

<GraphQLEditor
  query={`query getNumberOfDelegators($poolAddress: String) {
  num_active_delegator_per_pool(
    where: {pool_address: {_eq: $poolAddress}, num_active_delegator: {_gt: "0"}},
    distinct_on: pool_address
  ) {
    num_active_delegator
  }
}`}
  variables={`{
  "poolAddress": "0x06099edbe54f242bad50020dfd67646b1e46282999483e7064e70f02f7ea3c15"
}`}
/>

#### 変数:
- `$poolAddress`: **String** - アクティブな委任者数を照会したいプールのアドレス。例: `"0x06099edbe54f242bad50020dfd67646b1e46282999483e7064e70f02f7ea3c15"`
