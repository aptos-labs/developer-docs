# ホステッド型トランザクションストリームサービス

<IndexerBetaNotice />

[インデクサー API](../../indexer.mdx)の独自インスタンスや[インデクサー SDK](../indexer-sdk.mdx)のカスタムプロセッサーを実行している場合、トランザクションストリームサービスのインスタンスにアクセスする必要があります。このページでは、Aptos Labsがホストするトランザクションストリームサービスの使用方法について説明します。

## エンドポイント

特に指定がない限り、すべてのエンドポイントはGCP us-central1にあります。

- **メインネット:** grpc.mainnet.aptoslabs.com:443
- **テストネット:** grpc.testnet.aptoslabs.com:443
- **デブネット:** grpc.devnet.aptoslabs.com:443

## APIキーによる認証

Labs-Hostedトランザクションストリームサービスを使用するには、APIキーが必要です。APIキーを取得するには、以下の手順に従ってください：

1. https://developers.aptoslabs.com にアクセスします。
2. サインインし、左サイドバーの「API Keys」を選択します。
3. 新しいキーを作成します。最初のテーブルにトークン値が表示されます。

APIキーは`Authorization` HTTPヘッダーを設定することで提供できます（[MDN](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Authorization)）。curlを使用した例：

```bash filename="Terminal"
curl -H 'Authorization: Bearer aptoslabs_yj4donpaKy_Q6RBP4cdBmjA8T51hto1GcVX5ZS9S65dx'
```
