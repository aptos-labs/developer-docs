# パッケージの作成

<Callout>
開始する前にAptos CLIをインストールすることをお勧めします。
まだAptos CLIをインストールしていない場合は、[CLIセクション](../cli)をご覧ください。
</Callout>

<Steps>

### `aptos move init`

新しいプロジェクトディレクトリで、以下のコマンドを実行してMoveパッケージを初期化します：

```bash filename="Terminal"
aptos move init --name <PROJECT_NAME>
```

以下のような構造のMoveプロジェクトが作成されます：

</FileTree>

<Callout type={"default"}>
  [テンプレート](../cli/start-from-template)からMoveパッケージを作成することもできます。
</Callout>

### `Move.toml`の更新

`Move.toml`で、以下の重要な情報を入力します：

1. `name`: パッケージの名前
2. `version`: パッケージのバージョン（デフォルトは`"0.0.0"`）
3. `addresses`: モジュールがデプロイされるアドレスを記述します。これらは別名として使用できる名前付きアドレスです。以下の例では、`hello_blockchain`を名前付きアドレスとして使用します。
4. `dependencies`: `AptosFramework`やその他の[サードパーティの依存関係](./third-party-dependencies)を使用することになるでしょう。

以下は例です：

```toml filename="Move.toml"
[package]
name = "Examples"
version = "0.0.0"

[addresses]
hello_blockchain = "_"

[dependencies.AptosFramework]
git = "https://github.com/aptos-labs/aptos-framework.git"
rev = "mainnet"
subdir = "aptos-framework"
```
