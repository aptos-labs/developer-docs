# 整数

Move言語は、符号なし整数型として`u8`、`u16`、`u32`、`u64`、`u128`、`u256`をサポートしています。

| Type                             | Value Range              |
| -------------------------------- | ------------------------ |
| Unsigned 8-bit integer, `u8`     | 0 to 2<sup>8</sup> - 1   |
| Unsigned 16-bit integer, `u16`   | 0 to 2<sup>16</sup> - 1  |
| Unsigned 32-bit integer, `u32`   | 0 to 2<sup>32</sup> - 1  |
| Unsigned 64-bit integer, `u64`   | 0 to 2<sup>64</sup> - 1  |
| Unsigned 128-bit integer, `u128` | 0 to 2<sup>128</sup> - 1 |
| Unsigned 256-bit integer, `u256` | 0 to 2<sup>256</sup> - 1 |

## リテラル

数値リテラルは、10進数または16進数で記述できます：

```move
script {
  fun example() {
    // 10進数リテラル
    let d1 = 0;
    let d2 = 1337;

    // 16進数リテラル
    let h1 = 0x0;
    let h2 = 0x539;

    // 型注釈付きリテラル
    let val1: u8 = 0;
    let val2: u64 = 0;
    let val3: u128 = 0;
  }
}
```

型注釈がない場合、コンパイラはデフォルトで`u64`型を使用します。

### 数値の区切り文字

大きな数値を読みやすくするために、アンダースコア`_`を数値の区切り文字として使用できます：

```move
script {
  fun example() {
    let a = 123_456;
    let b = 123456; // aとbは等価
    let c = 1_2_3_4_5_6; // これも同じ値
    let d = 0xFF_FF_FF; // 16進数でも使用可能
  }
}
```

## 演算

### 算術演算

| 構文 | 説明 | 例 |
| ---- | ---- | --- |
| `+` | 加算 | `let sum = 1 + 2; // 3` |
| `-` | 減算 | `let difference = 3 - 1; // 2` |
| `*` | 乗算 | `let product = 2 * 3; // 6` |
| `%` | 剰余 | `let remainder = 5 % 2; // 1` |
| `/` | 除算 | `let quotient = 6 / 2; // 3` |

これらの演算子は、オーバーフローやアンダーフローが発生した場合、または0による除算が行われた場合にアボートします。

### ビット演算

| 構文 | 説明 | 例 |
| ---- | ---- | --- |
| `&` | ビット論理積（AND） | `let conj = 0x1 & 0x2; // 0x0` |
| <code>&vert;</code> | ビット論理和（OR） | <code>let disj = 0x1 &vert; 0x2; // 0x3</code> |
| `^` | ビット排他的論理和（XOR） | `let xor = 0x1 ^ 0x3; // 0x2` |
| `<<` | 左シフト | `let left = 1 << 2; // 4` |
| `>>` | 右シフト | `let right = 8 >> 1; // 4` |

### 比較演算

| 構文 | 説明 | 例 |
| ---- | ---- | --- |
| `<` | 未満 | `let less = 1 < 2; // true` |
| `>` | 超過 | `let greater = 2 > 1; // true` |
| `<=` | 以下 | `let leq = 1 <= 1; // true` |
| `>=` | 以上 | `let geq = 2 >= 1; // true` |

## キャスト

整数型間のキャストは、`as`演算子を使用して行います：

```move
script {
  fun example() {
    let x = 100u8 as u64;
    let y = 10000u64 as u128;
    let z = 10000u128 as u16;
  }
}
```

キャストは、値が目的の型の範囲内に収まらない場合にアボートします。

## 所有権

言語に組み込まれている他のスカラー値と同様に、整数値は暗黙的にコピー可能です。つまり、[`copy`](variables.mdx#move-and-copy)のような明示的な命令なしでコピーできます。
