# 変数

Move言語では、変数は`let`キーワードを使用して導入されます。すべての変数は、使用前に初期化する必要があります。

## 変数の宣言と初期化

変数は以下の方法で宣言できます：

```move
let x; // 型注釈なしの宣言
let y: u64; // 型注釈付きの宣言
let z = 42; // 宣言と初期化
let w: u64 = 42; // 型注釈付きの宣言と初期化
```

## 可変性

デフォルトでは、変数は不変です。変数を可変にするには、`mut`キーワードを使用します：

```move
let mut x = 0; // 可変変数
x = x + 1; // OK - xは可変
let y = 0; // 不変変数
y = y + 1; // エラー - yは不変
```

## シャドーイング

Move言語では、変数のシャドーイングが許可されています：

```move
let x = 0;
let x = x + 1; // OK - 新しいxが古いxをシャドーイング
```

## `move`と`copy`

Move言語では、値は一度だけ使用できます。値を複数回使用するには、明示的に`copy`する必要があります：

```move
let x = 10;
let y = copy x; // xの値をコピー
let z = move x; // xの値を移動（これ以降xは使用できません）
```

`copy`は以下の型でのみ使用できます：

- ブール値
- 整数
- アドレス
- 不変参照
- `copy`能力を持つ構造体

## スコープ

変数のスコープは、宣言されたブロックに限定されます：

```move
{
    let x = 0;
    {
        let y = x; // OK - xはこのスコープでアクセス可能
    }
    // yはここではアクセス不可
}
// xはここではアクセス不可
```

## アンダースコア変数

アンダースコア`_`は特別な変数で、値を無視するために使用できます：

```move
let _ = 0; // 値を無視
let (x, _) = (1, 2); // 2番目の値を無視
```

## 型推論

Move言語は、多くの場合、変数の型を推論できます：

```move
let x = 42; // u64型と推論
let y = &x; // &u64型と推論
```

ただし、場合によっては明示的な型注釈が必要です：

```move
let x: u8 = 42; // u8型を明示的に指定
```
