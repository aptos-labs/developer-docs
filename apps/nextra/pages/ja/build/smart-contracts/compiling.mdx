# コンパイル

<Callout>
コンパイルする前に、[CLI](../cli)が最新版であることを確認してください。
</Callout>

## `aptos move compile`

パッケージのセットアップが完了したら、以下のコマンドでMoveコードをコンパイルできます：

```bash filename="Terminal"
aptos move compile
```

正常に実行されると、以下のようなターミナル出力が表示されます：

```bash filename="Terminal"
{
  "Result": [
    "<PUBLISHING_ADDRESS>::<MODULE_NAME>"
  ]
}
```

<Callout>
  特に例を実行する場合、named addressesを追加する必要があるかもしれません。例えば、Hello Blockchain Moveの例では、`hello_blockchain` named addressを追加する必要があります：

  ```bash filename="Terminal"
  aptos move compile --named-addresses hello_blockchain=default
  ```
</Callout>

### `bytecode_modules`

bytecode_modulesフォルダには、モジュールのコンパイルされたMoveバイトコード（`module_name.mv`など）が含まれています。
バイトコードとそのセキュリティ機能の詳細については、[なぜMove?](./why-move)をご覧ください。

### `source_maps`

source_mapsフォルダには、ソースマップ（`module_name.mvsm`など）が含まれており、
コンパイルされたバイトコードをソースコードと関連する依存関係にマッピングすることができます。
