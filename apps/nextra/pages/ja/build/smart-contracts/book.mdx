# Move本

Moveへようこそ。Moveは、セキュアでサンドボックス化され、形式的に検証されたプログラミングのための次世代言語です。
Aptosを含む複数のブロックチェーンでスマートコントラクト言語として使用されています。
Moveを使用することで、開発者はアセットを柔軟に管理および転送するプログラムを作成でき、
同時にそれらのアセットに対する攻撃からの保護とセキュリティを提供します。
また、Moveはブロックチェーン以外のコンテキストでのユースケースも考慮して開発されています。

Moveは、[Rust](https://www.rust-lang.org/)からヒントを得て、通貨などのデジタルアセットを
明示的に表現するために、move（名前の由来）セマンティクスを持つリソース型を使用しています。

## Aptos Move本は誰のためのものか？

Moveは、セキュアで検証可能、かつ柔軟なプログラミング言語として設計・作成されました。
Moveの最初の使用はDiemブロックチェーンの実装のためであり、現在はAptosで使用されています。

この本は、プログラミングの経験があり、コアプログラミング言語の理解を深め、
その使用例を見たい開発者に適しています。
