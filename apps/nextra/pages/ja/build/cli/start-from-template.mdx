# テンプレートからMoveパッケージを開始する
以下の手順に従って、すぐに始めることができます。

<Steps>
  ### 初期化
  `hello-blockchain`テンプレートを使用してパッケージを初期化するには、以下を実行します：
  ```bash filename="Terminal"
  aptos move init --name hello_blockchain --template hello-blockchain
  ```

  ### 開発を始める
  テンプレートは`sources`ディレクトリの下に`hello_blockchain.move`ファイルを作成し、開発の開始をサポートします。
  <RemoteCodeblock
    permalink="https://github.com/aptos-labs/aptos-core/blob/afd3706c17bcccfb39a9d6059aecbfa648ed295d/aptos-move/move-examples/hello_blockchain/sources/hello_blockchain.move#L1-L64"
  />

  ### すべてのテンプレートを確認
  すべてのテンプレートを確認するには（およびパッケージの初期化に関する一般的なヘルプについては）、以下のコマンドを実行します：
  ```bash
  aptos move init --help
  ```

</Steps>
