# MacでのAptos CLIのインストール

Macの場合、Aptos CLIをインストールする最も簡単な方法は、パッケージマネージャー`brew`を使用することです。

# インストール

<Steps>
### `brew`がインストールされていることを確認してください https://brew.sh/
### 新しいターミナルを開き、以下のコマンドを入力します。

```bash filename="Terminal"
brew update
brew install aptos
```

### 別のターミナルを開き、`aptos help`を実行してCLIがインストールされていることを確認します。

```bash filename="Terminal"
aptos help
```
</Steps>

<Callout type="warning">
`brew`が動作しない場合は、こちらの手順を試してください：[特定バージョンのAptos CLIのインストール（上級者向け）](install-cli-specific-version.mdx)
</Callout>

# CLIのアップグレード

brewでCLIをアップグレードするには、2つのコマンドだけで済みます：

```bash filename="Terminal"
brew update
brew upgrade aptos
```
