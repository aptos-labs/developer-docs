# Aptos CLIを使用したネットワークノードの管理

[バリデータノードまたはバリデータフルノード（VFN）](../../network/nodes/validator-node.mdx)を実行している場合、CLIを使用してノードを操作することができます。

具体的には、以下のことができます：

1. [所有するステーキングプールの管理](../../network/nodes/validator-node/connect-nodes/staking-pool-operations.mdx)
2. [提案への投票](../../network/nodes/validator-node/connect-nodes/staking-pool-voter.mdx)

それ以外にも、ノードの操作に関連する特殊なコマンドについては、以下のヘルプコマンドを実行して確認できます：

```bash filename="Terminal"
aptos node --help
```
