# SolanaからAptosへのチートシート

違いと類似点についてさらに詳しく学ぶには[Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)をご覧ください。

|                              | Solana                                                                                                           | Aptos                                                                                                                                                              |
| ---------------------------- | ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **スマートコントラクト**          | Rust, SVM                                                                                                        | Move, MoveVM                                                                                                                                                       |
| **トランザクション手数料**         | 低額                                                                                                              | 低額                                                                                                                                                                |
| **並列処理**          | 悲観的並列処理、すべての書き込みアカウントを宣言する必要あり                                                      | 楽観的並列処理、チェーンが書き込みアカウントを推論                                                                                                        |
| **コントラクトアカウントサポート** | PDAアカウント                                                                                      | [オブジェクト](../smart-contracts/objects.mdx)または[リソースアカウント](../smart-contracts/resource-accounts.mdx)（オブジェクトの使用を推奨） |
| **データストレージ**             | プログラムが所有するアカウントにデータを保存                                                         | ユーザーアカウントまたはオブジェクト下のリソースとしてデータを保存                                                                                                               |
| **ストレージレベル**            | プログラムレベル                                                                                                    | オブジェクト下に保存する場合はグローバル                                                                                                                                    |
| **ストレージの考え方**          | ユーザーデータはアカウント下に分散して保存                                                                     | ユーザーデータはオブジェクト下に分散して保存                                                                                                                        |
| **コード例**             | [Solana上のTodoリストコントラクト](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Aptos上のTodoリストコントラクト](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos)                                                     |
| **呼び出し元ID**                | `signer`                                                                                                         | `signer`                                                                                                                                                           |
| **アップグレード可能性**            | プログラムはアップグレード可能                                                                            | モジュールはアップグレード可能                                                                                                                                               |
| **ディスパッチタイプ**            | 静的ディスパッチ                                                                                                  | 静的ディスパッチ                                                                                                                                                    |
| **FTスタンダード**             | トークンプログラム                                                                                                    | [Coin](../smart-contracts/aptos-coin.mdx)（レガシー）と[ファンジブルアセットスタンダード](../smart-contracts/fungible-asset.mdx)                                  |
| **NFTスタンダード**            | トークンプログラム                                                                                                    | [デジタルアセットスタンダード](../smart-contracts/digital-asset.mdx)                                                                                                |
| **ブロックチェーンとの対話**   | Solana web3.jsライブラリ                                                                           | [Aptos TypeScript SDK](../sdks/ts-sdk.mdx)                                                                                                 |
