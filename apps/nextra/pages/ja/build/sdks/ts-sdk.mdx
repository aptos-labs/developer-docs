TypeScript SDKを使用すると、Aptosブロックチェーンに接続し、探索し、対話することができます。データの要求、トランザクションの送信、テスト環境のセットアップなどが可能です！

```bash npm2yarn
npm i @aptos-labs/ts-sdk
```
