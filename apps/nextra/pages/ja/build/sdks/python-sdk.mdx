# Python SDK

Aptosは公式のPython SDKを軽度にメンテナンスしています。[PyPi](https://pypi.org/project/aptos-sdk/)で入手可能で、ソースコードは[aptos-python-sdk GitHubリポジトリ](https://github.com/aptos-labs/aptos-python-sdk)にあります。Python SDKの機能の多くは[Typescript SDK](ts-sdk.mdx)を反映しています。Python SDKの主な目的は、Python開発者がAptosに素早く慣れることを支援し、Aptosチュートリアルの補助として機能することです。

## Python SDKのインストール

Python SDKは`pip`経由、ソースから、または埋め込みでインストールできます：

### pipでインストール

`pip`経由でインストールするには：

```bash filename="Terminal"
pip3 install aptos-sdk
```

`aptos-sdk`はローカルのサイトパッケージディレクトリにインストールされます。例えば、macOSでは、`aptos-sdk`は`~/Library/Python/3.8/lib/python/site-packages/aptos_sdk`ディレクトリにあります。

### ソースコードからインストール

ソースからインストールするには：

```bash filename="Terminal"
git clone https://github.com/aptos-labs/aptos-python-sdk
pip3 install . --user
```

### 埋め込みでインストール

Python SDKを既存のPythonプロジェクトに埋め込むには：

```bash filename="Terminal"
cd /path/to/python/project
cp -r /path/to/aptos-python-sdk aptos-sdk
```
