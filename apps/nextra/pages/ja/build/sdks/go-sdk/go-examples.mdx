## コードスニペット

SDKリポジトリの[`examples`フォルダ](https://github.com/aptos-labs/aptos-go-sdk/tree/main/examples)には、
ニーズに合わせてカスタマイズできる多くのコードスニペットがあります。

### サンプルの実行方法

サンプルスクリプトを実行するには：
<Steps>
  ### [aptos-go-sdkリポジトリ](https://github.com/aptos-labs/aptos-go-sdk)をクローンします

  ```bash filename="Terminal"
  git clone https://github.com/aptos-labs/aptos-go-sdk.git
  ```

  ### パッケージのトップレベルから、すべての依存関係をインストールします。

  ```bash filename="Terminal"
  go install
  ```

  ### パッケージをビルドします。

  ```bash filename="Terminal"
  go build ./...
  ```

  ### サンプルを実行します

  ```bash filename="Terminal"
  go run examples/transfer_coin/main.go
  ```
</Steps>
