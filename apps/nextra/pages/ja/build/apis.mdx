# Aptos API

Aptosブロックチェーンネットワークには、ユースケースに応じて複数のAPIでアクセスできます。

## Aptosフルノード

フルノードに組み込まれているこのAPIは、Aptosブロックチェーンの状態を_読み取り_、トランザクションを_送信_するためのシンプルで低レイテンシーながら低レベルな方法を提供します。また、トランザクションのシミュレーションもサポートしています。

上記のAPIのコードはすべて[GitHub](https://github.com/aptos-labs/aptos-core)でオープンソース化されています。
そのため、誰でもこれらのAPIを運用することができ、世界中の多くの独立した運営者やビルダーがそれを選択しています。

### Aptos Labs が運用するAPIデプロイメント

[Aptos Labs](https://aptoslabs.com)は[Aptos Foundation](https://aptosfoundation.org/)に代わって、
各[Aptosネットワーク](../network/nodes/networks.mdx)のこれらのAPIのデプロイメントを運用し、一般に公開しています。

現在、Aptos LabsのAPIデプロイメントには2種類あります：

1. [匿名アクセスとIPベースのレート制限を持つAPI](../network/nodes/networks.mdx)
2. [[ベータ版] Aptos Labs開発者ポータルを通じた認証と開発者アカウントベースのレート制限を持つAPI](apis/aptos-labs-developer-portal.mdx)
