# Aptosのスマートコントラクト

Aptosのコントラクトは、複数のチェーンで使用される安全でサンドボックス化され、正式に検証されたプログラミングのための次世代言語であるMoveを使用して記述されます。
Moveを使用することで、開発者はアセットに対する攻撃から保護しながら、アセットを柔軟に管理および転送するプログラムを作成することができます。
