# Aptos APIs

Aptos 区块链网络可以通过多个 API 访问,具体取决于您的使用场景。

## Aptos 全节点

这个 API - 嵌入在全节点中 - 提供了一个简单、低延迟但低级别的方式来_读取_状态和_提交_交易到 Aptos 区块链。它还支持交易模拟。

## 索引器

<Cards>
   <Card href="./indexer">
      <Card.Title>索引器 GraphQL API</Card.Title>
      <Card.Description>
         这个 GraphQL API 提供了一个高级的、有主见的 GraphQL 接口来读取 Aptos 区块链的状态。
         它非常适合与 NFT、Aptos Objects 或自定义 Move 合约进行交互。
         在这里了解更多关于索引器驱动的 GraphQL API。
      </Card.Description>
   </Card>
   <Card href="./indexer/txn-stream">
      <Card.Title>交易流 API</Card.Title>
      <Card.Description>
         这个 GRPC API 将历史和实时交易数据流式传输到索引处理器。
         它被 Aptos Core Indexing 使用,也可以支持自定义的应用特定索引处理器
         用于实时区块链数据处理。在这里了解更多。
      </Card.Description>
   </Card>
</Cards>

上述每个 API 的代码都在 [GitHub](https://github.com/aptos-labs/aptos-core) 上开源。因此,
任何人都可以运行这些 API,全球许多独立运营商和构建者都选择这样做。

### Aptos Labs 运营的 API 部署

[Aptos Labs](https://aptoslabs.com) 代表 [Aptos Foundation](https://aptosfoundation.org/)
为每个 [Aptos 网络](../network/nodes/networks.mdx) 运营这些 API 的部署,并使其可供公众使用。

目前有 2 组 Aptos Labs API 部署:

1. [具有匿名访问和基于 IP 的速率限制的 API](../network/nodes/networks.mdx)
2. [[测试版] 通过 Aptos Labs 开发者门户进行身份验证和基于开发者账户速率限制的 API](apis/aptos-labs-developer-portal.mdx)
