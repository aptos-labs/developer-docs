# CLI

Aptos 命令行界面（CLI）是一个帮助您编译和测试 Move 合约的工具。它还可以帮助您快速体验链上 Aptos 功能。

对于高级用户，CLI 还可用于运行私有 Aptos 网络（以帮助本地测试代码），并有助于管理网络节点。
