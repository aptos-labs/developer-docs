# 命令行界面（CLI）

Aptos 命令行界面（CLI）是一个工具，用于帮助您编译和测试 Move 合约。它还可以帮助您快速体验 Aptos 在链上的特性。

对于更高级的用户，CLI 也可以用来运行私有的 Aptos 网络（以帮助在本地测试代码），并且可以有助于管理网络节点。
