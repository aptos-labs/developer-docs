# 通过 Aptos CLI 管理网络节点

如果你正在运行[验证节点或验证全节点(VFN)](../../network/nodes/validator-node.mdx)，你可以使用 CLI 与你的节点交互。

具体来说，你可以使用 CLI 来：

1. [管理你拥有的质押池](../../network/nodes/validator-node/connect-nodes/staking-pool-operations.mdx)。
2. [对提案进行投票](../../network/nodes/validator-node/connect-nodes/staking-pool-voter.mdx)。

除此之外，你可以运行以下帮助命令来查看 CLI 可以执行的更多与节点操作相关的专门命令：

```bash filename="Terminal"
aptos node --help
```
