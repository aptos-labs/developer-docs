# 通过 Aptos CLI 管理网络节点

如果您正在运行[验证者节点或验证者全节点（VFN）](../../network/nodes/validator-node.mdx)，您可以使用命令行界面（CLI）与您的节点交互。

具体来说，您可以使用 CLI 执行以下操作：
1. [管理您拥有的质押池](../../network/nodes/validator-node/connect-nodes/staking-pool-operations.mdx)。
2. [对提案进行投票](../../network/nodes/validator-node/connect-nodes/staking-pool-voter.mdx)。

除此之外，您可以运行以下帮助命令，查看更多与您的节点相关的执行特定的操作命令：

```bash filename="Terminal"
aptos node --help
```
