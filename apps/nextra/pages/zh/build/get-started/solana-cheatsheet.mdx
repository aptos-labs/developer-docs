# Solana 到 Aptos 速查表

如需了解更多差异与相似之处，请参阅 [Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)|                              | Solana                                                                                                           | Aptos                                                                                                                                                              |
| ---------------------------- | ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **智能合约**          | Rust, SVM                                                                                                        | Move, MoveVM                                                                                                                                                       |
| **交易费用**         | 低                                                                                                              | 低                                                                                                                                                                |
| **并行处理**          | 悲观并行，需要声明所有写入账户                                                      | 乐观并行，链自动推断写入账户                                                                                                        |
| **合约账户支持** | PDA账户                                                                                                      | [对象](../smart-contracts/objects.mdx) 或 [资源账户](../smart-contracts/resource-accounts.mdx)（推荐优先使用对象） |
| **数据存储**             | 数据存储在程序拥有的账户中                                                                         | 数据存储在用户账户或对象下的资源中                                                                                                               |
| **存储层级**            | 程序级别                                                                                                    | 全局存储（当使用对象时）                                                                                                                                    |
| **存储思路**          | 用户数据分布式存储在账户下                                                                     | 用户数据分布式存储在对象下                                                                                                                        |
| **示例代码**             | [Solana上的待办事项合约](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Aptos上的待办事项合约](https://github.com/aptos-lab请严格遵循系统指示翻译以下内容：

| **可升级性**            | 程序可升级                                                                                            | 模块可升级                                                                                                                                               |
| **调度类型**            | 静态调度                                                                                                  | 静态调度                                                                                                                                                    |
| **通证标准**             | Token 程序                                                                                                    | [Coin](../smart-contracts/aptos-coin.mdx) (旧版) 和 [可互换资产标准](../smart-contracts/fungible-asset.mdx)                                  |
| **非同质化通证标准**            | Token 程序                                                                                                    | [数字资产标准](../smart-contracts/digital-asset.mdx)                                                                                                |
