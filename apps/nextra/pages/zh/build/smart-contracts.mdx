# 智能合約

Aptos 的智能合約都是使用move語言,一種為安全為主的語言,沙盒和適合多鏈的正式驗證都是move語言的特色. 

Move 允許開發人員編寫靈活管理和轉移資產的程序，同時提供安全性和針對這些資產的攻擊的保護。
