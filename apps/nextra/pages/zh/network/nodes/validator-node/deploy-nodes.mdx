# 部署节点

在 Aptos 网络上运行验证节点和验证全节点（VFN）之前，你需要为你的节点选择一个部署方法。下面的指南提供了在各种平台和不同的 Aptos 网络上部署验证节点和 验证全节点（VFN）的逐步指导。

一旦你的节点部署完成，你可以通过初始化你的质押池和加入验证者集合将它们连接到 Aptos 网络。更多信息，请参见[连接节点](connect-nodes.mdx)。

<Callout type="warning">
在选择部署方法之前，请确保先阅读并理解[节点要求](node-requirements.mdx)。
</Callout>
