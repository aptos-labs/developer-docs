# 部署 PFN

以下指南提供了在 Aptos 网络上部署 PFN 的步骤。

<Callout type="info">
**我必须运行 PFN 吗？**<br />

如果您不想运行 PFN，但仍然想与 Aptos 区块链交互，您可以使用 Aptos Labs 提供的 REST API (见 [Aptos API](../../../build/apis.mdx))。不过，请注意，Aptos Labs 提供的 PFN 有速率限制，这可能会妨碍您的开发。如果您想避免这样的速率限制，您可以部署自己的 PFN 并与 Aptos 区块链直接同步。

</Callout>

<Callout type="warning">
**选择一个网络**<br />

这些指南默认将 PFN 部署在 Aptos 主网网络上，但它们也可以很容易地在其他网络（例如，devnet 或 testnet 网络）中执行相同的操作。在这样做时，请检出所需的网络分支，并使用相应分支的 `genesis.blob` 和 `waypoint.txt` 节点文件：
[主网](../configure/node-files-all-networks/node-files-mainnet.mdx)、
[devnet](../configure/node-files-all-networks/node-files-devnet.mdx) 和
[testnet](../configure/node-files-all-networks/node-files-testnet.mdx) 。

</Callout>
