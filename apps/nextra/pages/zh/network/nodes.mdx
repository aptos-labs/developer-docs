# 了解节点

Aptos 网络由三种类型的节点组成：验证者节点、验证者全节点（VFN）和公共全节点（PFN）。要参与共识，您需要运行一个验证者节点并质押最小数量的有效 Coin。VFN 和 PFN 不需要参与共识，但它们对于分发区块链数据和启用生态系统服务是必要的，例如索引、查询和 RESTful API 服务（见 [Aptos API](../build/apis.mdx)）。VFN 只能由验证者节点的运营者运行，而 PFN 可以由任何人运行。您可以在[深入理解区块链](blockchain.mdx)部分了解更多关于不同类型的节点。

本节提供了详细的分步指南，介绍如何在不同环境中部署和操作 Aptos 节点。它还描述了您需要进行质押、参与共识和治理的所有内容。
